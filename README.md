# 前端启动

## 有node.js运行环境

```css
cd client
```

```css
npm run start
```



## 无node.js运行环境

```css
cd dist
```

```css
在此目录开启一台web服务器
```



# 后端启动

```css
cd Server
```

```css
修改config.yml的数据库配置
```

## 有golang运行环境

```css
go run index.go
```

## 无golang运行环境(仅win)

```css
./index.exe
```
